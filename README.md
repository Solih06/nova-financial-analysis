# Nova Financial Solutions - Stock Sentiment Analysis

## Project Overview
This project focuses on analyzing financial news data to extract sentiments and correlate them with stock market movements. 

## Task 1: Exploratory Data Analysis (EDA)
- **Environment Setup:** Virtual environment and modular folder structure.
- **Analysis:** Headline length distribution, publisher activity, and time-series publication trends.
- **Key Finding:** Identified significant news spikes in 2020 correlating with global market volatility.

## Folder Structure
- `data/`: Raw and processed datasets.
- `notebooks/`: Jupyter notebooks for exploratory analysis.
- `scripts/`: Modular Python scripts for data processing.
